# 图标文件说明

这个目录应该包含Edge扩展的图标文件：

- icon16.png (16x16 像素)
- icon32.png (32x32 像素) 
- icon48.png (48x48 像素)
- icon128.png (128x128 像素)

你可以：
1. 使用在线图标生成器创建这些图标
2. 使用设计软件如Figma、Adobe Illustrator等创建
3. 从免费图标网站下载合适的图标

图标应该：
- 使用PNG格式
- 背景透明
- 简洁明了，易于识别
- 符合Microsoft Edge扩展的设计规范

如果暂时没有图标，扩展仍然可以正常工作，只是会显示默认图标。
