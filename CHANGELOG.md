# 更新日志

所有重要的项目变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.1] - 2024-08-07

### 修复 🐛
- 修复时间转换器中"现在"按钮未按照所选时区显示当前时间的问题
- 现在点击日期时间输入框旁的"现在"按钮会正确显示根据用户选择时区调整的当前时间

## [1.0.0] - 2024-08-07

### 新增 ✨
- 初始项目发布
- 实时时间戳显示功能
- 支持秒、毫秒、微秒三种精度
- Unix 时间戳与人类可读时间的双向转换
- 完整时区支持（UTC-12 到 UTC+12）
- 一键复制功能，点击即可复制时间戳或时间
- 现代化渐变 UI 设计，支持毛玻璃效果
- 智能输入验证和错误处理
- 响应式设计，适配不同屏幕尺寸

### 技术特性 🛠️
- React 18 + TypeScript 开发
- Webpack 5 模块化构建
- Chrome Extension Manifest V3
- 完整的类型安全保障
- 模块化组件架构

### 文件架构 📁
- 重新组织项目文件结构
- 提取时间戳处理工具函数
- 分离时区配置常量
- 优化组件代码复用

### 开发体验 👨‍💻
- 添加开发模式热重载
- TypeScript 类型检查
- 清晰的构建流程
- 完善的项目文档
